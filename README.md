Engineering Student Placement Analytics
Project Overview

This end-to-end data analytics project analyzes engineering student placement outcomes and identifies the factors that influence employability and placement success.

The project combines data cleaning, exploratory analysis, SQL querying, Python analysis, and Power BI dashboarding to generate actionable insights from student academic and skill-based data.

Objectives
Analyze placement trends among engineering students.
Identify key drivers of placement success.
Evaluate the impact of CGPA, internships, certifications, and technical skills.
Build an interactive dashboard for stakeholder decision-making.
